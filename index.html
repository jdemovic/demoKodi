Demolator Kodi Repository

Welcome to the official Demolator Kodi repository.

Install Instructions

  1. In Kodi, go to Settings → File manager.
  2. Add a new source: https://jdemovic.github.io/demoKodi/
  3. Go to Add-ons → Install from zip file.
  4. Select the source you added, then choose: repository.demolator-1.0.0.zip
  5. After installing the repository, go to Install from repository and choose the addon.
